Практична робота №10

Тема: Створення власних класів. Наслідування і віртуальні функції.
Мета: Навчитися створювати класи, нащадки, віртуальні функції та реалізовувати обчислення математичних виразів у вигляді деревоподібних структур.

Опис проекту

Репозиторій містить програму, розроблену у середовищі C++ Builder, для інтерпретації математичних виразів.
Програма дає змогу обчислювати вирази з операціями +, * та змінною x, використовуючи об’єктно-орієнтований підхід.

Структура

Unit1.cpp — основний код програми (класи, функції, події кнопок).

Unit1.h — заголовковий файл із описом компонентів.

ExprCalc.cbproj — проєкт C++ Builder.

Практична №10.pdf — звіт із виконаною роботою.
